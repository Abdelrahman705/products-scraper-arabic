# Arabic Products Scraper 🛍️

This project extracts Arabic product data from an eCommerce source and organizes it into a structured format.

### 📦 Collected Data:
- 🛒 Product Name  
- 💰 Price  
- 🏷️ Category  
- 📝 Description  
- 🖼️ Image URL (if available)

---

## 🛠️ Tools Used
- Python 🐍
- pandas
- BeautifulSoup
- requests

---

## 📁 Output
Saved as: `منتجات_منسقة.xlsx`
